# Quran Learning App (Baghdadi Qaida)

A modern, professional, and interactive Quran learning application built with Flutter.

## Features
- **Interactive Alphabet:** Learn Arabic letters with high-quality audio.
- **Baghdadi Qaida:** Structured lessons with word-by-word breakdown and offline audio.
- **Holy Quran:** Full mushaf support with verse-by-verse recitation (Offline).
- **Interactive Quizzes:** Test your knowledge with various question types.
- **Offline Mode:** All essential audio and text content are bundled within the app.
- **Multi-language Support:** English, Arabic, Dari, and Pashto.

## How to get the APK
This repository is configured with **GitHub Actions**. 
1. Go to the **Actions** tab in this repository.
2. Select the latest successful **Build APK** workflow.
3. Download the **app-release** artifact at the bottom of the page.

## Technical Details
- Framework: Flutter
- State Management: Riverpod
- Database: Hive (Local), Supabase (Optional Cloud Sync)
- Architecture: Clean Architecture
